# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2024-11-21

### Added
- Project list and detail screens with full CRUD operations
- Smart row/stitch counters that persist across app restarts
- Counter rename functionality
- Project status management (pause, resume, finish)
- Pattern library with catalog and import functionality
- Pattern Maker for creating custom pattern drafts
- Yarn stash management with color picker
- Project journal with photo timeline
- Custom accent color theming
- Light/dark/system theme support
- Data export/import functionality
- Cross-platform support (iOS, Android, Web)

### Fixed
- Project cards now display counter values correctly
- Yarn form starts empty when adding new entries
- Unified weight options across the app
- Removed cluttering UI elements from Patterns tab
- Improved spacing and iOS-native feel
- Fixed routing and navigation issues
- Real-time store updates

### Changed
- Cleaner, less overwhelming UI
- Improved navigation structure
- Better component organization
